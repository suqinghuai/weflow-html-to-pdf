# WEFLOW 聊天记录 HTML 转 PDF 插件

## 插件介绍

这是一个专门为 **WEFLOW 4.3.0** 版本微信聊天记录导出软件设计的插件，用于将导出的 HTML 格式聊天记录转换为 PDF 格式，方便打印和取证使用。

## 功能特点

- 📁 自动识别并处理同级目录下的所有 HTML 文件
- 🔪 智能拆分大文件，避免转换失败
- 🔧 自动修复 HTML 打印格式问题
- 📄 批量转换为 A4 格式 PDF 文件
- 🧹 自动清理临时文件，保持目录整洁
- 🎉 友好的终端界面，操作简单直观

## 系统要求

- Windows 10/11 操作系统
- 已安装 Microsoft Edge、Google Chrome 或 Chromium 浏览器之一
- WEFLOW 4.3.0 版本导出的 HTML 格式聊天记录

## 如何使用

### 步骤 1：准备工作

1. 使用 **WEFLOW 4.3.0** 导出微信聊天记录为 HTML 格式
2. 找到导出的 HTML 文件
3. 将 `HTML转PDF插件.exe` 文件复制到这个文件夹中
4. 双击运行 `HTML转PDF插件.exe` 

### 步骤 2：运行插件

1. 双击运行 `HTML转PDF插件.exe` 文件
2. 插件会自动检测当前目录下的 HTML 文件
3. 根据提示输入每个 PDF 文件包含的消息数量（默认 4000 条）

### 步骤 3：等待转换完成

插件会自动执行以下操作：
1. 🔪 拆分 HTML 文件（如果文件过大）
2. 🔧 修复 HTML 打印格式
3. 📄 转换为 PDF 文件
4. 🧹 清理临时文件

转换完成后，会在当前目录生成一个 `PDF输出` 文件夹，所有 PDF 文件都会保存在这里。

## 常见问题

### 问题：转换失败，提示没有找到可用浏览器

**解决方案**：
1. 确保已安装 Microsoft Edge、Google Chrome 或 Chromium 浏览器
2. 如果仍然失败，可以尝试运行以下命令安装浏览器：
   ```
   playwright install msedge
   # 或
   playwright install chrome
   # 或
   playwright install chromium
   ```

### 问题：生成的 PDF 文件不完整或格式不正确

**解决方案**：
1. 尝试减小每个文件包含的消息数量（建议 2000-3000 条）
2. 确保浏览器可以正常打开和渲染 HTML 文件

### 问题：程序运行后没有反应

**解决方案**：
1. 检查是否有其他程序占用了浏览器资源
2. 确保当前目录下有 HTML 文件
3. 尝试以管理员身份运行程序
